def decontracted(phrase):
    # specific
    phrase = re.sub(r"won't", "will not", phrase)
    phrase = re.sub(r"can\'t", "can not", phrase)

    # general
    phrase = re.sub(r"n\'t", " not", phrase)
    phrase = re.sub(r"\'re", " are", phrase)
    phrase = re.sub(r"\'s", " is", phrase)
    phrase = re.sub(r"\'d", " would", phrase)
    phrase = re.sub(r"\'ll", " will", phrase)
    phrase = re.sub(r"\'t", " not", phrase)
    phrase = re.sub(r"\'ve", " have", phrase)
    phrase = re.sub(r"\'m", " am", phrase)
    return phrase

df = raw.copy()

from tqdm import tqdm
preprocessed_reviews = []
# tqdm is for printing the status bar
for review in tqdm(df['reviews'].values):
    review = re.sub(r"&amp", '', review)
    review = re.sub(r"http\S+", "", review) # Removing http tags
    review = decontracted(review) # removing based on above functions
    review = re.sub("\S*\d\S*", " ", review).strip() # Removing alphanumeric 
    review = re.sub('[^A-Za-z]+', ' ', review) # removing special characters
    review = ' '.join(e.lower() for e in review.split() if e.lower() not in stopwords)
#     review = get_lemmatized(review)
    preprocessed_reviews.append(review.strip())

I. K-Means Clustering¶

The following k-means clustering uses five clusters following the optimal number of clusters identified in the internal validation criteria discussed in Notebook 2. Reviews per star rating underwent dimension reduction using TruncatedSVD at 80% variance explained. TSNE visualization of the results of the k-means clustering appear to have unparsimonious, unclear, unbalanced and less compact clusters relative to the results of the NMF clustering. Hence, k-means clustering was not considered as the model for our final clustering.

1. k-Means Clustering: Low-Rated Reviews¶

In [11]:
lsa = TruncatedSVD(int(bow_low.shape[1])-1)
low_lsa = lsa.fit_transform(bow_low)
var = lsa.explained_variance_ratio_
c_var = var.cumsum()
n_comp = np.searchsorted(lsa.explained_variance_ratio_.cumsum(), 0.8) + 1

lsa = TruncatedSVD(n_components = n_comp)
low_lsa = lsa.fit_transform(bow_low)

kmeans = KMeans(n_clusters=5, max_iter=1000, algorithm = 'auto')

kmeans_transform_low = kmeans.fit_transform(low_lsa)
tsne_low = TSNE(random_state=1208).fit_transform(kmeans_transform_low)

prediction = kmeans.fit_predict(low_lsa)
plt.scatter(tsne_low[:, 0], tsne_low[:, 1], c=prediction)
plt.xlabel('Principal Component 1')
plt.ylabel('Principal Component 2')
plt.title('t-SNE Visualization of Low-rated Reviews');

II. Agglomerative Clustering: Complete Linkage Method¶

Below are results of the agglomerative clustering of Amazon reviews using the complete linkage method. The dendograms show 4, 5, and 5 clusters for low-rated, neutral-rated and high rated reviews, respectively. TSNE visualization of the clusters show no distinct, compact and balanced clustering. Hemce, agglomerative clustering was not considered as the best model for extracting themes from amazon.com reviews.
